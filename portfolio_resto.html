Analyse de comportements des utilisateurs d'un Serious Game

J'ai effectué cette mission pour un client de SCALIAN Alyotech



  • Contexte :

    • Pour augmenté l'efficacité des agents de la gestion des situations des fuites de gaz



  • Objectif :

    • Développement d'IA de communication, Analyse et modélisation de comportement des utilisateurs d'un Serious Game
    en réalité virtuelle (environnement virtuel contenant des simulations des situations de fuite de gaz)



  • Tâches effectuées :

    • - Collecte des audios des appels téléphonique depuis le server du client
    - Transcription automatique des audio en texte avec chacune des librairies suivantes :

    1

    		CMU Sphinx,

    2

    		Google API Speech Recognition,

    3

    		Google Cloud Speech API ,

    4

    		IBM Speech to Text (IBM Watson Studio)
    - Evaluation de la performance de trancription de chacun de ces librairies
    - Contruction d'une base de données textuelles.
    - Développement des intelligences artificielles (IA) de communication en réalité virtuelle
    - Visualisation du parcours des utilisateurs dans l'environnement virtuelle
    - Modélisation du parcours des utilisateurs en EV
    - Evaluation de la performance des utilisateurs



  • Résultats :

    • Parmis les quatre API de Transcription des audio en textes,
    seule Google API était le meilleur Le CMU Sphinx présentait unn avantage considérable.
    Car il n'utilise pas la connection internet contrairement aux autres.
    Les utilisateurs utilisant fréquemment le logiciel de la RV reste plus efficatité (...).



  • Technologies utilisées :

    • Python, Jupyter, Unity, Google API
